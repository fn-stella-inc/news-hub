## Features

- AI-Ready Content Architecture: Designed to render content generated by AI systems
- HTMX Integration: Dynamic content loading without full page refreshes
- Category Filtering: Browse news by technology, science, business, and culture
- Infinite Scroll: Load more articles seamlessly with HTMX
- Responsive Design: Optimized for all device sizes
- Dark Theme: Modern dark interface with accent colors
- SEO Optimized: Complete meta tags, Open Graph, Twitter Cards, and structured data
- Accessibility: WCAG compliant with proper ARIA labels and keyboard navigation
- Performance: Static site generation with optimized assets

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm 9.x or higher (or pnpm/yarn)
- Docker and Docker Compose (optional, for containerized deployment)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/news-hub.git
cd news-hub
```

2. Install dependencies:

```bash
npm install
```

3. Copy the environment file:

```bash
cp .env.example .env
```

4. Configure your environment variables in `.env`

### Development

Start the development server with hot reload:

```bash
npm run dev
```

The application will be available at `http://localhost:4321`

### Production Build

Build the static site for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Docker Deployment

### Using Docker Compose (Recommended)

Start the application in production mode:

```bash
docker compose up -d
```

Start in development mode with hot reload:

```bash
docker compose --profile development up
```

Start with nginx reverse proxy:

```bash
docker compose --profile production up -d
```

### Using Docker Directly

Build the image:

```bash
docker build -t news-hub .
```

Run the container:

```bash
docker run -d -p 4321:4321 --name news-hub news-hub
```

## Configuration

### Environment Variables

| Variable            | Description                       | Default                 |
| ------------------- | --------------------------------- | ----------------------- |
| `SITE_URL`          | Public URL of the site            | `http://localhost:4321` |
| `SITE_NAME`         | Name displayed in metadata        | `News Hub`              |
| `AI_API_KEY`        | API key for AI content generation | -                       |
| `AI_API_URL`        | Endpoint for AI content API       | -                       |
| `ENABLE_AI_CONTENT` | Enable AI content features        | `true`                  |
| `ENABLE_NEWSLETTER` | Enable newsletter subscription    | `true`                  |
| `NODE_ENV`          | Environment mode                  | `development`           |

### Astro Configuration

Modify `astro.config.mjs` to customize:

- Site URL and base path
- Build output settings
- Integrations (sitemap, etc.)
- Vite configuration

## API Endpoints

### GET /api/articles

Fetch paginated articles with optional filtering.

Query Parameters:

| Parameter  | Type   | Default | Description        |
| ---------- | ------ | ------- | ------------------ |
| `page`     | number | 1       | Page number        |
| `limit`    | number | 6       | Articles per page  |
| `category` | string | all     | Filter by category |

Response (JSON):

```json
{
  "articles": [...],
  "page": 1,
  "hasMore": true,
  "category": "all"
}
```

HTMX Response (HTML):

Returns pre-rendered article cards for direct DOM insertion.

## SEO Features

News Hub includes comprehensive SEO optimization:

- Meta Tags: Title, description, keywords, robots
- Open Graph: Full OG tag support for social sharing
- Twitter Cards: Large image card support
- Structured Data: JSON-LD schema for WebSite and Article types
- Sitemap: Auto-generated XML sitemap
- Canonical URLs: Proper canonical tag implementation
- Semantic HTML: Proper heading hierarchy and landmarks

### Meta Tag Configuration

Configure in `base-layout.astro`:

```astro
<BaseLayout
  title="Page Title"
  description="Page description for SEO"
  keywords={['keyword1', 'keyword2']}
  ogImage="/custom-image.png"
  ogType="article"
/>
```

## Customization

### Theme Colors

Modify CSS custom properties in `src/styles/global.css`:

```css
:root {
  --color-accent-primary: #ff6b35;
  --color-accent-secondary: #f7c59f;
  --color-accent-tertiary: #2ec4b6;
}
```

### Typography

Change fonts in `base-layout.astro` and update CSS variables:

```css
:root {
  --font-display: "Your Display Font", serif;
  --font-body: "Your Body Font", sans-serif;
}
```

### Adding Categories

1. Add category to filter in `category-filter.astro`
2. Add color class in `article-card.astro`
3. Update sample data in `lib/articles.ts`

### AI Content Integration

Replace sample data with your AI content source:

1. Modify `src/lib/articles.ts` to fetch from your AI API
2. Update API endpoint in `src/pages/api/articles.ts`
3. Configure `AI_API_URL` and `AI_API_KEY` in `.env`
